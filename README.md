# AI冷蔵庫シェフ (AI Refrigerator Chef) 🍳

冷蔵庫の中身を写真に撮るだけで、AI（Gemini）が現在の材料に最適なレシピを提案してくれる、お料理初心者専用のWebアプリです。

## ✨ 特徴
- **写真で簡単判別**: 複数の写真を解析して食材を特定。
- **初心者向け設定**: 人数（1〜4人前）を選択でき、分量や単位（g, cc, 大さじ等）を明確に表示。
- **柔軟なレシピ提案**: 写真に写った食材を「すべて」使うのではなく、最適な組み合わせをAIが選択。
- **北欧モダンデザイン**: 清潔感のあるキッチンをイメージした、使いやすいUI。

## 🛠 テクニカルスタック
- **Frontend**: React (ES Modules), Tailwind CSS, Lucide React
- **AI**: Google Gemini 2.0 Flash (Google GenAI SDK)
- **Deployment**: Netlify / Vercel など

## 🚀 使い方
1. アプリを起動し、人数を選択します。
2. 「カメラを起動 / 写真を選択」ボタンから冷蔵庫の中を撮影（複数枚OK）。
3. 「レシピを生成」ボタンを押すと、AIシェフがレシピを考案します。

## ⚠️ セキュリティについて
このアプリは環境変数 `process.env.API_KEY` を使用します。
GitHubにコードを上げる際は、**APIキーをコード内に直接書き込まないように**注意してください。

## 🛠 開発者向けセットアップ
1. このリポジトリをクローンします。
2. Google AI Studio で Gemini API キーを取得します。
3. デプロイ先（Netlifyなど）の環境変数に `API_KEY` を設定してください。
